# Git for project management

This is an example Git repository that is used as part of the Git for project management workshop.

The aim of this course is to demonstrate how Github can be used to manage your project and collaborate with others on a project effectively .

For the purposes of this course we will consider building a bungalow with the following plan:

1 - Site Preparation: Clearing and levelling the plot

2 - Foundations: Excavating trenches, pouring concrete, installing drainage, and constructing the damp-proof course (DPC)

3 - Super structure: Build walls and roof

4 - Make weathertight: Fit windows and doors

5 - First fix:  Installing internal stud walls, plumbing pipes, electrical wiring, and heating systems before plastering.

6 - Plastering: Covering interior walls and ceilings. 

7 - Second fix: Installing finished electrical items (sockets/switches), plumbing (taps/boiler), doors, and kitchen/bathroom fittings.

8 - Final finishes: Decorating, flooring, landscaping
